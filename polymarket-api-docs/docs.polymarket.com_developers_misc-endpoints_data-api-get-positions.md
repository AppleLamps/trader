#### Query Parameters

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-user)

user

string

required

User address (required)
User Profile Address (0x-prefixed, 40 hex chars)

Example:

`"0x56687bf447db6ffa42ffe2204a05edaa20f55839"`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-market)

market

string\[\]

Comma-separated list of condition IDs. Mutually exclusive with eventId.

0x-prefixed 64-hex string

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-event-id)

eventId

integer\[\]

Comma-separated list of event IDs. Mutually exclusive with market.

Required range: `x >= 1`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-size-threshold)

sizeThreshold

number

default:1

Required range: `x >= 0`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-redeemable)

redeemable

boolean

default:false

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-mergeable)

mergeable

boolean

default:false

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-limit)

limit

integer

default:100

Required range: `0 <= x <= 500`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-offset)

offset

integer

default:0

Required range: `0 <= x <= 10000`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-sort-by)

sortBy

enum<string>

default:TOKENS

Available options:

`CURRENT`,

`INITIAL`,

`TOKENS`,

`CASHPNL`,

`PERCENTPNL`,

`TITLE`,

`RESOLVING`,

`PRICE`,

`AVGPRICE`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-sort-direction)

sortDirection

enum<string>

default:DESC

Available options:

`ASC`,

`DESC`

[​](https://docs.polymarket.com/developers/misc-endpoints/data-api-get-positions#parameter-title)

title

string

Maximum string length: `100`
