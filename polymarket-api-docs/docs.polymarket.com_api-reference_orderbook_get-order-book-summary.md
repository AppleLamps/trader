string

required

string

required

Asset identifier

`"1234567890"`

timestamp

string<date-time>

required

`"2023-10-01T12:00:00Z"`

hash

string

required

Hash of the order book state

`"0xabc123def456..."`

bids

required

asks

object\[\]

Showchild attributes

string

required

`"0.001"`

string

required
